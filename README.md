# javascript-gestore-file

consegna del prof: 
Realizzare una applicazione Web che:
- legge il contenuto di un file di testo fileInput.txt
- lo presenta a video in una textArea
- permetta di modificarne il contenuto
- previa pressione di un pulsante dedicato, scriva il contenuto della textArea in un nuovo file fileOutput.txt.
Il due file di testo sono gestiti da un server Node.

Vengono forniti:
- schermata.jpg: la schermata dell'applicazione da realizzare
- script.js: la codifica javascript dell'applicazione
- server.js: la codifica del server NODE
- fileInput.txt: esempio di file di input
